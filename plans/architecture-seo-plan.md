# Архитектура и текстовый прототип сайта веб-студии

## 1. Анализ ниши и Семантическое ядро

### Целевая аудитория и позиционирование
- **Аудитория:** Частные лица (эксперты, фрилансеры) и микробизнес.
- **Ценовой сегмент:** Доступный (от 5000 рублей), но с визуальным позиционированием "премиум".
- **Тональность (Tone of Voice):** Дружелюбная, прозрачная, экспертная, без сложного IT-жаргона. Фокус на выгоде для бизнеса и надежности.

### Семантическое ядро (СЯ)

**Высокочастотные запросы (ВЧ):**
- заказать сайт
- создание сайтов
- разработка мобильных приложений
- сайт под ключ

**Среднечастотные запросы (СЧ):**
- заказать сайт визитку
- создание сайта недорого
- разработка приложения для ios и android
- сделать сайт для малого бизнеса
- услуги веб разработчика
- стоимость создания сайта

**Низкочастотные запросы (НЧ - "длинный хвост" для конверсии):**
- заказать сайт визитку за 5000 рублей
- разработка мобильного приложения для микробизнеса
- создание сайта с современным дизайном недорого
- разработка сайта без предоплаты для частных лиц

## 2. Логическая структура сайта (5 страниц)

1. **Главная страница (Home)** - концентрация основных ВЧ запросов, хаб для перехода на остальные страницы.
2. **Услуги и Цены (Services)** - СЧ и НЧ запросы коммерческой направленности (создание сайта недорого, разработка приложений).
3. **Портфолио (Portfolio)** - подтверждение экспертизы (E-E-A-T), запросы связанные с примерами работ.
4. **О студии (About)** - фокус на доверии, факторах E-E-A-T (экспертность, авторитетность, достоверность).
5. **Контакты (Contact)** - локальные запросы, конверсионный этап.

## 4. Визуальные решения и UI-компоненты (Next.js)

### Эстетика Apple (Гайдлайны)
- **Шрифты:** San Francisco Pro (SF Pro Display / Text) или Inter. Использование трекинга и контраста весов (Bold для заголовков, Regular для набора).
- **Негативное пространство (Whitespace):** Огромные отступы (padding/margin) между секциями. Контент должен "дышать".
- **Сетка и компоновка:** Строгая симметрия или выверенная асимметрия с крупными заголовками, выровненными по левому краю или центру.
- **Цвета (Светлая тема):** Фон `#F5F5F7` или `#FFFFFF`, текст `#1D1D1F`, акценты приглушенно синие `#0066CC` или монохром.
- **Цвета (Темная тема):** Фон `#000000`, карточки `#1C1C1E`, текст `#F5F5F7`.
- **Микроинтеракции:** Плавные появления элементов при скролле (Framer Motion), эффект матового стекла (backdrop-filter: blur) для навигации.

### UI-компоненты для Next.js
1.  **`Navigation Menu (Header)`:**
    - Фиксированное сверху, полупрозрачное (`backdrop-blur-md`).
    - Минимум пунктов, плавное подчеркивание при наведении.
2.  **`Hero Section`:**
    - Крупный типографический H1, короткий подзаголовок, две кнопки (Primary - заливка, Secondary - outline).
    - Плавная анимация появления (fade-in, slide-up).
3.  **`Feature Cards (Услуги/Преимущества)`:**
    - Карточки с легким скруглением углов (`rounded-2xl` или `3xl`).
    - В темной теме — тонкий border (`border-gray-800`), в светлой — едва заметная тень (`shadow-sm`).
    - Иконки минималистичные, линейные (например, Lucide React).
4.  **`Theme Toggle Switch`:**
    - Изящный переключатель Sun/Moon в шапке.

### Логика переключения тем оформления (Next.js + Tailwind CSS)
Рекомендуется использовать `next-themes` совместно с Tailwind CSS (конфигурация `darkMode: 'class'`).

```tsx
// Пример базовой логики с next-themes
import { useTheme } from 'next-themes'
import { useEffect, useState } from 'react'

export const ThemeToggle = () => {
  const { theme, setTheme } = useTheme()
  const [mounted, setMounted] = useState(false)

  // Избегаем hydration mismatch
  useEffect(() => setMounted(true), [])
  if (!mounted) return null

  return (
    <button
      onClick={() => setTheme(theme === 'dark' ? 'light' : 'dark')}
      className="p-2 rounded-full bg-gray-200 dark:bg-gray-800 transition-colors duration-300"
      aria-label="Toggle Dark Mode"
    >
      {theme === 'dark' ? '☀️' : '🌙'}
    </button>
  )
}
```

### Генерация изображений (Концепт)
Вместо заглушек должны использоваться сгенерированные или стоковые фото премиум-качества:
- **Hero Image (Главная):** Абстрактный 3D-рендер стеклянных слоев или минималистичного смартфона, парящего в воздухе на чистом фоне.
- **Портфолио:** Аккуратные мокапы (iPhone 15 Pro, MacBook Air) с вставленными внутрь интерфейсами, расположенные под углом с мягкими тенями.
- **О нас:** Размытый фон светлого офиса с фокусом на текстуру (например, часть клавиатуры или блокнот), создающий атмосферу рабочего процесса без стоковых лиц.
