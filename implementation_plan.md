# Implementation Plan

## Задача
Привести тексты сайта к более современному и аккуратному русскоязычному UI-стилю:
- заменить подпись в портфолио на «Проекты, за которые нам не стыдно»;
- проверить орфографию и оформление заголовков;
- убрать лишние точки в коротких заголовках и оставить конечные знаки в обычных абзацах там, где они нужны.

## Основание
- Для коротких заголовков интерфейса и секционных тайтлов точка в конце обычно не ставится.
- Для обычных абзацев и полноценных описаний конечные знаки сохраняются.
- Русские и смешанные термины лучше приводить к единообразной записи: `веб-разработка`, `iOS-разработка`, `Android-разработка`, `UX/UI-дизайн`, `релиз`.

## Внешние ориентиры
- Microsoft Style Guide: короткие UI-элементы и заголовки оформляются без лишней конечной пунктуации.
- Русские справочники по оформлению заголовков: в конце заголовков точка не ставится.
